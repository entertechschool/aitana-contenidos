Grilla mensual

Inicio de mes

Antes de producir cualquier pieza, planificas el mes completo. Pegas la estrategia del mes junto al prompt y Claude genera todas las ideas. Tú curadas, ajustas y cargas en ClickUp.

Importante: Siempre pega la estrategia del mes al inicio del prompt — sin eso Claude no puede personalizar las ideas.
Prompt — Grilla mensual
[Pega aquí la estrategia del mes antes de enviar]

Necesito la grilla de contenido para [mes] de Enter Tech School.

Formatos: carruseles educativos, reels, flyers de pauta.
Frecuencia: 4-5 posts semanales.
Balance: 70% educativo / 30% conversión.

Para cada pieza dame:
- Tipo de formato
- Programa o tema
- Título o ángulo propuesto
- Objetivo (awareness / educación / conversión)
- Fecha sugerida

Nada genérico. Cada pieza debe sentirse 100% Enter.
2A

Carruseles educativos

HTML → editar → capturar → Shots → Canva
Prompt — Carrusel educativo
Crea un carrusel educativo para Instagram de Enter Tech School.

Tema: [qué vas a enseñar]
Programa: [Desarrollo Web / Análisis de Datos / Ciberseguridad / Soluciones con IA / Enter general]
Objetivo: [awareness / educación / conversión]
Slides: [un número, o escribe "a tu criterio"]
Ángulo (opcional): [idea o frase de partida]

Entrégalo como archivo HTML con el sistema de diseño Enter.
↔ Desliza para ver los pasos en orden
1
Claude entrega el código HTML
Cuando Claude termina, verás un bloque de código. En la esquina de ese bloque hay un botón que dice Copiar. Presiónalo — el código se copia solo.
2
Pega en el HTML Viewer
Ve a html.onlineviewer.net. Pega el código en el cuadro de texto y presiona "Preview". Verás todos los slides del carrusel.
Usa la vista amplia para ver cada slide en su tamaño real.
3
Revisar hasta que esté perfecto
Revisa diseño, textos y colores. Si algo está mal en el diseño, cuéntale a Claude qué está mal — él lo corrige. Si es solo un texto, ve al paso 4.
Correcciones de texto → hazlas tú. Correcciones de diseño → pídele a Claude.
4
Editar textos con Ctrl+F
Guarda el archivo HTML. Ábrelo en el Bloc de Notas. Presiona Ctrl+F (Windows) o Cmd+F (Mac). Busca la palabra → reemplázala → guarda → vuelve a previsualizar.
Solo cambia el texto. Nunca borres las partes con < o > — esas controlan el diseño.
5
Captura slide por slide
Con todo listo, toma captura de cada slide:
• Mac: Cmd+Shift+4
• Windows: Win+Shift+S
Selecciona el área exacta del slide.
Captura al 100% de zoom, sin ampliar ni reducir la pantalla.
6
Shots.so — fondo techie
Entra a shots.so. Sube la captura. Elige el fondo según el color del programa (tabla abajo). Descarga. Repite por cada slide.
7
Canva + @handle + exportar
Pega las imágenes en Canva. Añade @entertechschool. Exporta como JPG o PNG.
✳ Recién aquí le pides el siguiente carrusel a Claude — cuando terminaste este proceso completo.

Copies en batch

Al final de cada sesión
Por qué al final y juntos: Pedir los copies de varias piezas en un mensaje gasta muchos menos tokens que pedirlos uno por uno. Y Claude mantiene mejor el tono cuando los ve todos en contexto.
Prompt — Copies batch
Dame los copies de los últimos [número] carruseles/reels que produjimos.

Para cada pieza:
- Copy Instagram (hook + desarrollo + CTA + hashtags)
- Copy TikTok (versión corta y directa)
- Copy LinkedIn (tono profesional)

Cuándo editarlo tú vs. pedírselo a Claude

Cada mensaje que le mandas a Claude usa tokens (tiene un costo). La regla de oro: cambios de texto o números → hazlos tú. Cambios de diseño o estructura → pídele a Claude.

✓ Edítalo tú (gratis, 30 seg)
  • Cambiar una palabra o frase
  • Corregir una tilde o tipeo
  • Ajustar tamaño de texto (50px → 44px)
  • Cambiar un número o dato
  • Modificar el CTA o la URL
  • Cambiar el nombre de una herramienta
✗ Pídele a Claude (úsalo bien)
  • El diseño está mal (halos, colores, layout)
  • Hay un error de información importante
  • El copy necesita reescritura real
  • Quieres un concepto diferente
  • Necesitas añadir o quitar un slide entero
1
Sé específico desde el primer mensaje — Cuanto más contexto das, menos correcciones necesitas. Incluye siempre: tema, programa, objetivo, número de slides.
2
Pide variaciones juntas — "Dame 3 opciones de título" en un prompt, no 3 mensajes separados.
3
Copies siempre al final y en batch — Al terminar la sesión de producción, pídelos todos de golpe en un solo mensaje.
4
No repitas contexto — Claude recuerda toda la conversación. No le expliques de nuevo quién es Enter si ya está en el proyecto.
